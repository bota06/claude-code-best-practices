# Lessons Learned

このファイルは過去のミスから抽出したルールを蓄積する。
ミスが発生したら「Reflect → Abstract → Generalize」してここに追記すること。

---

## 追記の手順

```
1. Reflect:  なぜこのミスが起きたか（具体的な状況）
2. Abstract: このミスの本質（一般化できる形で）
3. Generalize: 次回から守るべきルール
4. Write:    このファイルに [Lxxx] として追記
```

---

## テンプレート

### [L001] タイトル
**ミス**: 何が起きたか（具体的に）。
**ルール**: 次回から守るべきルール（一般化して記述）。

### [L002] タイトル
**ミス**: ...
**ルール**: ...
